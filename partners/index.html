협력 방식(공식)

협력은 “로고 교환”이 아니라, 운영 구조를 공유하는 것입니다.

공동 프로그램
교육/훈련/캠페인 공동 운영
기준·심사 협력
기준 설계, 심사위원 참여
기관 인증
운영 기준 충족 기관 인증
조회 연동
코드/검증 구조 공동 적용
협력 문의하기 기준·규정 인증 안내

협력기관 목록(표기 영역)

실제 협력기관을 확정하면 아래 “데이터”에 추가만 하면 자동 반영됩니다. (기관 공개는 동의 범위 내에서만 진행하세요.)

※ 카드에 로고 이미지를 넣고 싶으면 assets/images/에 로고 파일을 올린 뒤 데이터에 경로만 넣으면 됩니다.

협력 요건(요약)

  • 공익 목적과 운영 방향에 동의
  • 기준·윤리·안전 원칙 준수
  • 필요 시 문서/현장 확인 협조
  • 조회(검증) 구조 적용 시 데이터 정확성 책임

함께 만들 파트너를 기다립니다

협력 제안은 “어떤 프로그램을, 어떤 대상에게, 어떤 방식으로 운영할지”를 간단히 적어주시면 됩니다. 재단은 기준·절차·검증 구조로 운영을 지원합니다.
