기준 운영 선언

국제공익자격인증재단은 공익 목적의 자격·인증이 사회에서 신뢰를 얻도록, 평가 요건과 판정 기준을 문서화하고, 절차에 따라 심사하며, 조회(검증) 서비스로 결과를 확인할 수 있도록 운영합니다.

표준화
동일 기준·동일 절차·동일 기록 방식
추적 가능
심사 기록과 상태 변경 이력 관리
검증 가능
코드 기반 조회로 유효성 확인
자격·인증 조회 서비스 서식/규정 다운로드

심사 기준 프레임(고급)

어떤 인증이든 아래 5개 축을 기준으로 설계·평가합니다. (분야별 세부 항목은 별도 기준서로 운영)

1) 목적
공익성/필요성/범위
2) 역량
지식/기술/태도
3) 안전
안전기준/리스크관리
4) 윤리
인권/공정/책임
5) 검증
기록/재현/조회

판정 및 상태 정의

상태 의미 대표 사유 후속 조치
VALID 유효기간 내 정상 상태 기준 충족, 갱신 완료 조회 서비스 표시
EXPIRED 유효기간 종료 갱신 미완료 갱신 신청 권고
SUSPENDED 일시 정지 규정 미준수, 보완 필요 보완 후 재심
REVOKED 효력 폐기 허위 제출, 중대 위반 재발급 제한/재심 절차
